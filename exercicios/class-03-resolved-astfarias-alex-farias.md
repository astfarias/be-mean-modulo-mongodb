## 4 - Liste todos Pokemons com o name `Pikachu` **OU** com attack **menor ou igual que** 0.5
```
    var query1 = {name : "Pikachu"}
    var query2 = {attack : {$lte : 0.5}}
    db.pokemons.find({$or : [query1, query2]})
```
